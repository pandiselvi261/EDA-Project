# Exploratory Data Analysis (EDA)

## Objective
To perform Exploratory Data Analysis using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Dataset
Built-in Iris Dataset from Scikit-learn.

## Tasks Performed
- Loaded dataset
- Explored dataset structure
- Displayed statistical summary
- Checked missing values
- Checked duplicate values
- Created histograms
- Created box plot
- Created scatter plot
- Generated correlation matrix

## Conclusion
The dataset was successfully analyzed using Exploratory Data Analysis techniques. Visualizations and statistical summaries helped understand the dataset.
